# Last Stand - Tienda de suministros tácticos (Proyecto final)
## ¿Porque last Stand?

La idea surgio mientras jugaba al Project Zomboid, juego de supervivencia zombie en el que por obvias razones los suministros militares son cruciales para la supervivencia, durante otras actividades escolares, especificamente en la creacion de una base de datos, recorde esto mismo, por lo que la base de datos la base en esta tienda ficticia y posteriormente la adapte para este proyecto de desarrollo web, se que esta informacion es algo irrelevante pero queria compartirla.

## Temática

Tienda especializada en suministros militares, tácticos y de supervivencia, en la que se ofrecen cosas como armamento, accesorios, equipo de protección, munición, raciones y más.

## Instrucciones de uso del filtro (página Equipo táctico)

1. Ve a la página **Equipo táctico** (`serviciosExpress.html`).
2. En la barra de búsqueda escribe el nombre del producto (ej. "chaleco") o una categoría (ej. "Munición").
3. Haz clic en el botón **Buscar** o presiona **Enter**.
4. La cuadrícula mostrará solo los productos que coincidan con tu búsqueda.
5. Para ver detalles de un producto, haz clic en **Ver detalles** (se abre un modal con información completa).
6. Para reiniciar la vista, borra el texto y vuelve a buscar (o refresca la página).

## Páginas del sitio

- **Inicio** Pantalla principal con dos carruseles de imágenes destacadas.
- **Quiénes somos**  Misión, visión y valores.
- **Armería** Catálogo estático de 18 armas populares con especificaciones y compatibilidad.
- **Equipo táctico** Catálogo dinámico con 20 productos (accesorios, munición, equipo, raciones) y filtro de búsqueda.
- **Contacto** Formulario con validación de campos e información de contacto directo.
- **Políticas de Privacidad** Politicas de privacidad.
- **Términos y Condiciones** Terminos y condiciones.
